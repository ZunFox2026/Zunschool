# Làm quen với Python
## Giới thiệu
Python là một ngôn ngữ lập trình cấp cao, dễ học và được sử dụng rộng rãi trong nhiều lĩnh vực như phát triển web, phân tích dữ liệu, trí tuệ nhân tạo, v.v. Python được tạo ra vào cuối những năm 1980 bởi Guido van Rossum và được phát hành vào năm 1991. Từ đó, Python đã trở thành một trong những ngôn ngữ lập trình phổ biến nhất thế giới.

Python có nhiều ưu điểm như dễ học, dễ sử dụng, có thể chạy trên nhiều nền tảng khác nhau (Windows, macOS, Linux), và có một cộng đồng lập trình viên lớn và tích cực. Python cũng có nhiều thư viện và framework hỗ trợ cho việc phát triển ứng dụng, như Django, Flask, NumPy, Pandas, v.v.

## Lý thuyết
Python là một ngôn ngữ lập trình hướng đối tượng, nghĩa là nó tổ chức code thành các đối tượng và các phương thức hoạt động trên các đối tượng đó. Python cũng hỗ trợ các khái niệm lập trình khác như lập trình chức năng, lập trình thủ tục, v.v.

Trong Python, chúng ta có các loại dữ liệu cơ bản như số nguyên (int), số thực (float), chuỗi (str), danh sách (list), từ điển (dict), v.v. Chúng ta cũng có các cấu trúc điều khiển như if-else, for, while, v.v.

Python cũng hỗ trợ các khái niệm như hàm (function), lớp (class), đối tượng (object), v.v. Hàm là một khối code có thể được gọi nhiều lần với các tham số khác nhau. Lớp là một định nghĩa cho một đối tượng, và đối tượng là một thể hiện của lớp đó.

## Ví dụ
Dưới đây là một ví dụ đơn giản về việc sử dụng Python:
```python
# In ra màn hình
print("Xin chào, thế giới!")

# Khai báo biến
x = 5
y = 10

# In ra giá trị của biến
print("Giá trị của x là:", x)
print("Giá trị của y là:", y)

# Sử dụng if-else
if x > y:
    print("x lớn hơn y")
else:
    print("x nhỏ hơn hoặc bằng y")

# Sử dụng for
fruits = ["táo", "chuối", "cam"]
for fruit in fruits:
    print(fruit)
```
Ví dụ trên minh họa cách sử dụng các loại dữ liệu cơ bản, cấu trúc điều khiển, và các khái niệm lập trình khác trong Python.

* Viết một chương trình Python sử dụng for để in ra các phần tử của một danh sách.
