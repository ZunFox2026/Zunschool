# Python 57: Lập Trình Cơ Bản Cho Người Mới Bắt Đầu

## Giới thiệu

Chào mừng bạn đến với bài học thứ 57 trong chuỗi bài học Python dành cho người mới bắt đầu! Bài học này cung cấp nền tảng cơ bản nhất để bạn bắt đầu hành trình lập trình với Python – một trong những ngôn ngữ phổ biến và dễ học nhất hiện nay. Dù bạn chưa từng viết dòng code nào, chỉ cần có máy tính và tinh thần học hỏi, bạn hoàn toàn có thể làm được. Mục tiêu của bài học là giúp bạn hiểu được cấu trúc cơ bản của một chương trình Python, cách viết và chạy code đơn giản.

## Lý thuyết

Lập trình là quá trình đưa ra các chỉ dẫn cho máy tính thực hiện công việc. Trong Python, bạn bắt đầu bằng các câu lệnh đơn giản như in ra màn hình, khai báo biến, hoặc thực hiện phép tính. Một chương trình Python thường bắt đầu từ câu lệnh `print()` để hiển thị nội dung. Bạn cũng sẽ làm quen với các kiểu dữ liệu cơ bản như số nguyên (`int`), số thập phân (`float`), và chuỗi ký tự (`str`). Ngoài ra, Python sử dụng thụt lề (indentation) để xác định khối lệnh thay vì dấu ngoặc như các ngôn ngữ khác.

## Ví dụ

Dưới đây là một ví dụ đơn giản in lời chào và thực hiện phép cộng:

```python
# In lời chào
print("Xin chào, mình là Python!")

# Tính tổng hai số
a = 5
b = 3
tong = a + b
print("Tổng của", a, "và", b, "là:", tong)
```

Kết quả khi chạy chương trình:
```
Xin chào, mình là Python!
Tổng của 5 và 3 là: 8
```
