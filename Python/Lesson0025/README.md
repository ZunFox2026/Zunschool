# Bài học Python số 25 - Cấp độ Trung cấp

## Chủ đề: Xử lý ngoại lệ (Exception Handling) và thiết kế khối `try-except` nâng cao

Trong bài học này, chúng ta sẽ tìm hiểu sâu hơn về cách xử lý lỗi trong Python bằng cơ chế ngoại lệ (exception handling). Bạn sẽ học cách bắt lỗi một cách thông minh, xử lý nhiều loại lỗi riêng biệt, sử dụng các khối `else` và `finally`, cũng như tự tạo ra ngoại lệ tùy chỉnh. Đây là kỹ năng quan trọng để viết code ổn định và chuyên nghiệp hơn.

### Mục tiêu bài học
- Hiểu được cách hoạt động của cơ chế ngoại lệ trong Python.
- Biết cách sử dụng `try`, `except`, `else`, và `finally` một cách hiệu quả.
- Xử lý nhiều loại ngoại lệ khác nhau.
- Tạo và sử dụng ngoại lệ tùy chỉnh.
- Áp dụng xử lý ngoại lệ vào các tình huống thực tế như đọc file, nhập liệu người dùng.

### Hướng dẫn sử dụng
1. Mở file `main.py` để xem các ví dụ minh họa.
2. Làm bài tập trong file `exercises.py`.
3. Kiểm tra lời giải tại `solutions.py`.

Chạy các file bằng lệnh:
```bash
python main.py
python exercises.py  # (sau khi hoàn thành bài tập)
```
