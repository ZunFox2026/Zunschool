# Bài 42: Python Cơ bản

## Giới thiệu

Python là một ngôn ngữ lập trình mạnh mẽ, dễ học và được sử dụng rộng rãi trong nhiều lĩnh vực như phát triển web, phân tích dữ liệu, trí tuệ nhân tạo và tự động hóa. Bài học này sẽ cung cấp những kiến thức cơ bản nhất về Python, giúp người mới bắt đầu làm quen với cú pháp, cấu trúc và cách viết chương trình đơn giản.

Python nổi bật nhờ cú pháp rõ ràng, dễ đọc và khả năng chạy trên nhiều nền tảng khác nhau. Với triết lý "đơn giản là tốt hơn", Python rất phù hợp để học lập trình từ những bước đầu tiên.

## Lý thuyết

Một chương trình Python bắt đầu từ các câu lệnh đơn giản. Dưới đây là một số khái niệm cơ bản:
- **Biến**: Dùng để lưu trữ dữ liệu. Ví dụ: `ten = "An"`.
- **Kiểu dữ liệu**: Python hỗ trợ các kiểu như số nguyên (`int`), số thực (`float`), chuỗi (`str`), và giá trị logic (`bool`).
- **In dữ liệu**: Dùng hàm `print()` để hiển thị thông tin ra màn hình.
- **Nhập dữ liệu**: Dùng hàm `input()` để người dùng nhập thông tin từ bàn phím.

Python không yêu cầu khai báo kiểu dữ liệu trước, và sử dụng thụt lề (indentation) để phân chia khối lệnh thay vì dấu ngoặc.

## Ví dụ

Dưới đây là chương trình Python đơn giản yêu cầu người dùng nhập tên và chào họ:

```python
# Nhập tên từ người dùng
ten = input("Nhập tên của bạn: ")

# In lời chào
print("Xin chào,", ten)
```

Khi chạy, chương trình sẽ hiển thị:  
`Nhập tên của bạn: `  
Sau khi người dùng gõ tên và nhấn Enter, chương trình in ra:  
`Xin chào, [tên người dùng]`

Gợi ý: Sử dụng `int(input(...))` để chuyển chuỗi thành số nguyên.
